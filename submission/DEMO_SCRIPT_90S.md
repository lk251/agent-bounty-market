verification, and settlement are hard to trust.

and protected verifier receipt decide acceptance. Candidate-owned CI is not
authoritative.

portion and retained operating credit. The fresh split Stripe transfer, real
GitHub lifecycle, Nemotron-backed Hermes decisions, and OpenShell/NemoClaw run
remain blocked or fallback in this bundle.

budgeted, verified, replay-safe work.
