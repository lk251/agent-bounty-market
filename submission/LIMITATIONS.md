# Limitations

- Not legal escrow.
- Not a production public marketplace.
- No arbitrary autonomous spending: trusted policy gates every budget movement.
- Local demo uses deterministic fake providers and says so visibly.
- Prior real Stripe sandbox evidence covers a full-transfer settlement loop,
  not the new split-retain-spend external transfer.
- Real GitHub credentials/webhook, Hermes runtime wrapper, and OpenShell/NemoClaw
  backend are not configured in this environment.
- No authenticated recorded-real bundle exists yet for the complete GitHub +
  Hermes/NVIDIA + Stripe split-retain-spend presentation.
