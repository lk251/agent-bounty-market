# Limitations

- Not a regulated custody product.
- Not a production public marketplace.
- No arbitrary autonomous spending: trusted policy gates every budget movement.
- The current winning bundle is `Mixed real/fallback` and says so visibly.
- Deterministic fallback providers are still used for blocked live components.
- Prior real Stripe sandbox evidence covers a full-transfer settlement loop,
  not the new split-retain-spend external transfer.
- Real GitHub credentials/webhook, Nemotron-backed Hermes wrapper,
  Docker/OpenShell/NemoClaw backend, and fresh Stripe split-transfer env are not
  configured in this process.
- No all-sponsor live bundle exists yet for the complete GitHub +
  Hermes/NVIDIA + Stripe split-retain-spend presentation.
- Retained operating credit is an internal ledger liability governed by trusted
  policy, not an external account controlled by an agent.
